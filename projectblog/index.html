Entry 1: Starting Out

Starting my programming journey was both exciting and challenging. I decided to begin with HTML and CSS because they form the foundation of web development. Learning the basics of HTML tags, elements, and CSS properties was a great first step.

Every day I learned something new, like how to structure web pages and style text. Seeing my first simple web page was so satisfying, and I felt motivated to keep learning more.

Entry 2: My First Project

For my first project, I decided to create a personal portfolio page. I included sections for my biography, skills, and a few projects I want to showcase. Creating this page helped me practice both my HTML structure skills and my CSS styling.

I spent time learning how to use different font styles, colors, and layout techniques. This project gave me the confidence to start learning more complex topics like JavaScript!
